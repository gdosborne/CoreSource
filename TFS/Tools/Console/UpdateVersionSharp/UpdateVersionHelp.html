UpdateVersionSharp Help

The UpdateVersionSharp Tool (UpdateVersion) provides an automatic, configurable way for a developer to manage the versions of their assemblies.

Versioning information
Command Line Parameters
Parameter Value
/e Display Error Codes
/p Project file name

Usage Example1: /p="test.csproj"
Usage Example2: /p="$(ProjectName)"

The available schema methods are defined below:

Method Description
Day Two-digit day value
DayValueFrom Number of days from a specific parameter date
Ignore Leaves value unchanged
Month Two-digit current month value
Random A random value
SecondValueFrom Number of Seconds from a specific parameter date
Year Four-digit current year value
TwoDigitYear Two-digit current year value
Increment Increments the current value by 1
IncrementEachDay Increments the current value by 1 the first build of the day, afterwords leaves value unchanged
IncrementResetEachDay Increments the current value by 1 and resets the value to zero the first build of the day
